## Ex1 (12pts) :

### Initialiser the project :

```bash
> npm init --y
```

### Requirements :

```bash
> npm init --y
> npm install express
> npm i joi
> npm i nodemon
> npm i mongoose
> npm i debug
> npm i config
> npm i bcryptjs
> npm i jsonwebtoken
```

### Commits of source control :

```bash
> git log --oneline
```

### Run the project :

```bash
> npm start
```

### NB :

> You can migrate the connection to your MongoDB !
